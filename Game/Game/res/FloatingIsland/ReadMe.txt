■コリジョンメッシュのネーミングルール
　「UCX_[表示用メッシュ名]」または「UCX_[表示用メッシュ名]_[任意の数字]」になります。
　表示用メッシュに対しコリジョンメッシュは複数存在する可能性があります。
　[任意の数字]の大小はコリジョン処理に影響は及ぼしません。
　例： Bridge.MV1
　　　　+ Bridge 表示用メッシュ
　　　　+ UCX_Bridge_1　コリジョンメッシュ
　　　　+ UCX_Bridge_2　コリジョンメッシュ
　　　　+ UCX_Bridge_3　コリジョンメッシュ
　　　　+ UCX_Bridge_4　コリジョンメッシュ
　　　　+ UCX_Bridge_5　コリジョンメッシュ
　　　　+ UCX_Bridge_6　コリジョンメッシュ
　　　　+ UCX_Bridge_7　コリジョンメッシュ
　　　　+ UCX_Bridge_8　コリジョンメッシュ
　　　　+ UCX_Bridge_9　コリジョンメッシュ
